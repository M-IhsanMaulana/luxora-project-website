# Documentation & Versioning Rules

You MUST always refer to the following official documentation when generating code or providing technical advice for this Luxora Bot brand website.

### Official Documentation Links:
- **Next.js (App Router):** https://nextjs.org/docs
- **React 19:** https://react.dev/reference/react
- **Tailwind CSS v4:** https://tailwindcss.com/docs
- **Framer Motion:** https://www.framer.com/motion/
- **TypeScript:** https://www.typescriptlang.org/docs/

### Core Instructions:
1. **Version Compliance:** Ensure you use Next.js App Router guidelines and NOT the deprecated Pages Router structure.
2. **Server vs Client Components:** Pay close attention to when a component needs `"use client"` based on React 19 and Next.js docs.
3. **Styling:** Follow the official Tailwind structure for utility building.
